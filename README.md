아무 이유도 모른 채 음산한 독방에 수감된 플레이어블 캐릭터를 탈출시키면 종료되는 프로그램입니다.<br/>
난이도는 크게 어렵지 않고 독백으로 힌트 아닌 힌트를 넣어둬 쉽게 클리어할 수 있습니다.<br/>

개발을 차차 배워가고 있어 미숙하고 엉성한 부분이 많지만 양해 부탁드리겠습니다.<br/><br/>
